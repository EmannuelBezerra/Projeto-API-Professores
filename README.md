# 📚 Projeto API REST - Professores

Desenvolvimento de uma API REST utilizando arquitetura MVC com **Node.js**, **Express** e **MySQL**.

---

## 🎯 Objetivo

Desenvolver uma API REST completa, com foco em boas práticas de desenvolvimento,
organização de projeto e uso correto de programação assíncrona. Esse projeto foi feito com base no exemplo fornecido em sala, desenvolvendo a API - Clientes.

---

## 🛠️ Tecnologias Utilizadas

- ![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat&logo=nodedotjs&logoColor=white)
- ![Express](https://img.shields.io/badge/Express-000000?style=flat&logo=express&logoColor=white)
- ![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql&logoColor=white)
- Arquitetura **MVC**
- Programação assíncrona com **async/await**

---

## 📋 Aspectos Desenvolvidos

- ✅ Organização do projeto
- ✅ Entendimento da arquitetura MVC
- ✅ Rotas REST
- ✅ Manipulação de banco de dados relacional
- ✅ Uso correto de programação assíncrona
- ✅ Boas práticas básicas
